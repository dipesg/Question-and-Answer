## Installation and running

*  Clone the repository
```
git clone https://github.com/dipesg/Question-and-Answer.git
```

*  Create and activate conda environment:
```
conda create -n venv python = 3.9 -y
conda activate venv
```

* Run the app:
```
python app.py
```
